# WellnessBot 🤖

Telegram-бот для ежедневного отслеживания самочувствия и продуктивности

## Структура проекта

```
.
├── main.py          # Точка входа: инициализация бота, планировщик, polling
├── handler.py       # Все обработчики команд и колбэков (роуты)
├── keyboards.py     # Все клавиатуры (ReplyKeyboard и InlineKeyboard)
├── database.py      # Подключение к БД и все SQL-методы
├── graph.py         # Генерация графиков через matplotlib
├── schema.sql       # DDL: создание таблиц и индексов
├── test_data.sql    # Тестовые данные (15 записей)
├── requirements.txt # Зависимости с версиями
├── .env.example     # Пример файла переменных окружения
└── .gitignore
```

## 🎮 Кнопки меню


| Кнопка                         | Описание                                                                                                   |
| :----------------------------------- | :----------------------------------------------------------------------------------------------------------------- |
| **➕ Записать день**     | Пошаговый ввод данных: настроение → работа → сон → комментарий |
| **📊 Статистика**          | Просмотр статистики за неделю/месяц, инсайты и график                 |
| **📜 История**                | Показать последние 5 записей                                                               |
| **🧹 Очистить данные** | Удалить всю историю (требуется подтверждение)                               |
| **👩‍💻 Помощь**              | Показать список команд                                                                         |
